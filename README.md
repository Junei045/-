# あいろの タイピングきょうしつ

小学校向けタイピング練習Webアプリと、先生用の指導ガイドです。
飯島小中学校学校運営協議会／豊田連合の AI 教育教材の一部として作成しました。

## 構成

| ファイル | 内容 | 対象 |
|---|---|---|
| [`typing-school.html`](./typing-school.html) | 児童用タイピング練習アプリ（Webブラウザで動作） | 児童 |
| [`typing-guide.html`](./typing-guide.html) | 指導ガイド（Web表示・印刷対応） | 先生 |
| [`typing-guide.pdf`](./typing-guide.pdf) | 指導ガイド（PDF版・印刷用） | 先生 |

## 使い方

### 児童用アプリ（typing-school.html）

ブラウザで開くだけで使えます。インターネット接続がない環境でも動作します。

- GitHub Pages で公開している場合：URLを開く
- ファイルを配布する場合：`typing-school.html` をダブルクリックして開く

トップ画面で学年（低学年／中学年／高学年）を選び、第1回〜第3回から練習を選びます。

- **低学年（1・2年）**：ローマ字は使わず、キーの場所・アルファベットに親しむ
- **中学年（3・4年）**：ローマ字入力の基礎（かな→ことば）
- **高学年（5・6年）**：タッチタイピング・文章入力・スピードチャレンジ

各回の終わりに ★（最大3つ）・正確さ・1分間の文字数が表示されます。

### 先生用ガイド（typing-guide.html / typing-guide.pdf）

各回（全9コマ・1コマ45分）の授業の流れ、声かけ例、つまずき対応をまとめています。
巻末に印刷用の「がんばりカード」付きです。

- PDF版：ダウンロードしてそのまま印刷
- HTML版：ブラウザで開いて `Ctrl+P` で印刷／PDF保存も可能

## 動作環境

- 対応：PC・Chromebookの一般的なブラウザ（Chrome、Edge等）
- 児童用アプリはインターネット接続不要（フォント表示のみオンライン時に最適化）
